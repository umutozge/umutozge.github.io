    Plotting

    The most basic way to create plots in Python is by using the matplotlib library. Below is an example of how to create a simple line plot.

    import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]

plt.plot(x, y)
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.title("Simple plot")
plt.show()
